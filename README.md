# voice_local_agent

**전용 하드웨어 버튼을 누르면 듣고, 한국어로 대화하며, PC를 제어하는 완전 로컬 음성 비서**

음성 인식·대화·음성 합성이 모두 **이 PC 안에서** 처리됩니다. 대화 내용이 외부 AI 서버로 전송되지 않습니다. (웹 검색·유튜브 등 웹에 접속하는 기능만 인터넷을 사용합니다.)

> ⚠️ 이 저장소는 **배포(다운로드) 전용**입니다. 소스 코드는 공개되지 않습니다.
> 설치 파일은 [Releases](../../releases)에서 받을 수 있습니다.

---

## 주요 기능

- 🎙 **전용 HW 버튼 PTT** — 버튼을 누르는 동안 듣고, 놓으면 실행합니다.
- 🧠 **완전 로컬 AI** — 음성·대화가 이 PC에서만 처리됩니다(오프라인 추론).
- 🖥 **PC 제어** — 앱 실행, 창 제어, 파일 검색·정리, 클립보드, 시스템 볼륨·미디어 키, 전원 제어 등.
- 🌐 **웹 작업** — 유튜브 검색·재생, 구글·네이버 검색, 페이지 이동·클릭 등.
- ⏰ **일상 도구** — 날짜·시간 안내, 타이머·알림.
- 🔒 **안전장치** — 삭제·종료·명령 실행 등 되돌리기 어려운 작업은 실행 전 재확인합니다.

---

## 시스템 요구사항

| 항목 | 최소 / 권장 |
|------|-------------|
| OS | **Windows 11** (64-bit) |
| 실행 방식 | **전용 하드웨어 장치** 필요 (음성 기능은 정품 장치 연결 시 동작) |
| RAM | 8GB 이상(권장 16GB+) |
| GPU | 선택 — NVIDIA GPU가 있으면 더 빠른 모델을 자동 선택, 없으면 CPU로 동작 |
| 저장공간 | 설치 약 0.5GB + **AI 모델 약 2~5GB**(첫 실행 시 다운로드) |
| 인터넷 | 첫 실행 시 모델 다운로드에 필요. 이후 AI 처리는 오프라인 |
| 브라우저 | 웹 기능 사용 시 **Google Chrome** 설치 필요 |

설치 환경(GPU/RAM)에 맞춰 음성 인식·언어 모델의 크기가 **자동으로 선택**됩니다.

## 설치

1. [Releases](../../releases)에서 최신 `voice_local_agent_setup.exe`를 내려받습니다.
2. 실행 후 안내에 따라 설치합니다(설치 중 사용자 라이선스 계약 동의가 필요합니다).
3. 처음 실행하면 **초기 설정 마법사**가 열립니다:
   - 마이크 접근·PC 제어 권한 동의
   - AI 모델 준비(자동 다운로드 — 네트워크 속도에 따라 수 분 소요)
   - 브라우저·하드웨어 안내
4. 전용 장치를 연결하면 음성 명령을 사용할 수 있습니다.

> 💡 설치 파일은 현재 코드 서명이 적용되지 않아, Windows SmartScreen이
> "알 수 없는 게시자" 경고를 표시할 수 있습니다. **추가 정보 → 실행**으로 진행하세요.

---

## 사용법

1. 전용 하드웨어의 버튼을 **누른 채** 말합니다.
2. 버튼을 **놓으면** 명령이 실행됩니다.
3. 답변은 화면과 음성으로 전달됩니다.

예시:
- "메모장 열어줘"
- "유튜브에서 로파이 음악 틀어줘"
- "다운로드 폴더에서 보고서 찾아줘"
- "10분 뒤에 알려줘"

창을 닫아도 트레이에서 계속 실행됩니다. 완전 종료는 트레이 아이콘 메뉴에서 할 수 있습니다.

## 개인정보 처리

- 음성·대화 텍스트는 **이 PC에만** 존재하며 외부 AI로 전송되지 않습니다.
- 웹 검색·유튜브 등 **웹에 접속하는 기능**은 해당 사이트로의 일반적인 인터넷 요청을 발생시킵니다.
