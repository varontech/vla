## v0.1.0 — 프로토타입

- **Tag**: `v0.1.0`
- **Title**: voice_local_agent 0.1.0 — 프로토타입
- **Pre-release**: ✅

전용 하드웨어 버튼을 누르면 듣고, 한국어로 대화하며, PC를 제어하는 **완전 로컬 음성 비서**입니다.
음성 인식·대화·음성 합성이 모두 이 PC 안에서 처리되며, 대화 내용은 외부 AI 서버로 전송되지 않습니다.

> ⚠️ **프로토타입(0.1.0)** 입니다. 평가·시연 목적으로 배포하며, 상용 판매본이 아닙니다.

### 📥 설치

1. 아래 **Assets**에서 `voice_local_agent_setup.exe`를 내려받아 실행합니다.
2. 설치 중 **사용자 라이선스 계약(EULA)** 동의가 필요합니다.
3. 첫 실행 시 **초기 설정 마법사**가 열립니다 — 권한 동의 → AI 모델 자동 다운로드(2~5GB, 네트워크 속도에 따라 수 분) → 준비 완료.
4. 전용 장치를 연결하면 음성 명령을 사용할 수 있습니다.

> 💡 코드 서명 전이라 Windows SmartScreen이 **"알 수 없는 게시자"** 경고를 표시할 수 있습니다. **추가 정보 → 실행**으로 진행하세요.

### 🖥 시스템 요구사항

| 항목 | 내용 |
|------|------|
| OS | Windows 11 (64-bit) |
| RAM | 8GB 이상 (권장 16GB+) |
| 저장공간 | 설치 약 0.5GB + AI 모델 2~5GB(첫 실행 다운로드) |
| GPU | 선택 — 현재 빌드는 CPU 추론 |
| 음성 기능 | **전용 하드웨어 장치** 연결 시 동작 |
| 웹 기능 | Google Chrome 설치 필요 |

### ✨ 주요 기능

- 🎬 **유튜브** — 열기·검색·재생, "아무거나 틀어줘", 뒤로가기
- 🔎 **웹 검색** — 구글·네이버 검색
- 🖥 **PC 제어** — 앱 실행·종료, 볼륨·미디어, 창 제어
- 📁 **파일** — 폴더 열기·파일 검색
- 🕐 **시간** — 현재 시각·날짜(한국 기준)
- ⚡ **빠른 응답·말 끊기(barge-in)·명령 취소** 지원, 앱 내 **사용법 탭** 제공
- 🔒 삭제·종료 등 되돌리기 어려운 작업은 실행 전 재확인

### ⚠️ 알려진 제한

- **전용 하드웨어 장치**가 있어야 음성/명령 기능이 활성화됩니다.
- 코드 서명이 적용되지 않아 SmartScreen 경고가 표시됩니다.
- AI 추론이 CPU 기준이라, 사양에 따라 첫 응답이 느릴 수 있습니다(시작 시 자동 준비).
